Icecream

Github Link : https://github.com/izbaatif/Icecream

Icecream is a simple responsive webpage containing information about an ice cream parlour such as menu and deals.

The entire html part is written using jsx inside react.js component function and jquery is used for the menu's see more and see less functionality whereas the slideshow is made using bootstrap.

Technologies used :

  • HTML
  • CSS
  • Bootstrap
  • React.js
  • jQuery
