Products Database

Github Link : https://github.com/izbaatif/productsDatabase

View Report : https://docs.google.com/document/d/1DmOhB_yEgqpOJc4OfcB4VN_3iQnUR8FYYN2SJwWcmTY/edit?usp=sharing

This is a project in which the csv tables were first imported into mysql and then normalised. It includes a website with the pages buy products which is accessible to anyone and when the button is pressed to buy item, it adds it to the orders table. The other two pages, add products or edit products is only available to the admin of the website and they use the mySQL crud funcionalities.
