InterrogatorX

Github Link : https://github.com/izbaatif/interrogatorX

View report : https://docs.google.com/document/d/1c53bgSWOvr37Tf3HsVyxM4_MbwBv3EvDqE_retjyPyY/edit?usp=sharing

InterrogatorX: This is a straightforward web application designed for the classification of news articles as either genuine or fake. It comprises two input fields that enable users to input the title of the news article and the content within the article. Users can also choose from one of three available options in the dropdown menu (explained further below) to determine if the news article is real or fake.

Data Collection and Model Training: In the initial phase, a substantial dataset for training the model on fake news was obtained from Kaggle. The model employs three distinct feature extraction techniques represented by the available dropdown options, which are: Bag of Words, TF-IDF Vectorization, and Word2Vec. For each feature extractor, the machine learning algorithm that yielded the highest accuracy was selected. Subsequently, these models were saved and exported into pickle files.

Integration and Implementation: The pickle files containing the trained models and data were then imported into the Python backend, which was developed using Flask. When a user enters data into the web application, this information is passed to the respective models to provide a functional and user-friendly interface.

Technologies used :

  • HTML
  • CSS
  • Flask
  • Data Analysis using python
  • AI using Python
